This project is a React TypeScript Vite web app representing a Fish Tank simulation.
Key directories:
- `src/engine/`: Core logic for simulating creatures, collisions, movement, reproduction, running in a 60fps loop independent of React state.
- `src/renderer/`: Rendering logic for drawing creatures and the tank on the canvas.
- `src/ui/`: React components for UI interactions like color palette, brush picker, and creation panel.
- `src/hooks/`: Custom React hooks, including `useDrawingCanvas` for the canvas drawing interface.
- `src/store/`: Zustand global state management.
